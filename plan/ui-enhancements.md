# UI/UX 改进

## 现状分析

当前前端基于 Vue 3 + TypeScript，使用 Wails v2 运行时。主要组件：

- `MainLayout.vue`：主布局（侧边栏 + 内容区）
- `ChatPanel.vue` / `MessageBubble.vue`：对话界面
- `FileExplorer.vue` / `FileTransfer.vue`：文件浏览和传输
- `PlanPanel.vue` / `TodoBoard.vue`：计划模式展示
- `SettingsPanel.vue`：设置面板（SSH/Docker/LLM/MCP 配置）
- `TerminalPanel.vue`：终端面板
- 固定暗色主题，无自定义选项
- 无快捷键系统

---

## 改进方向

### 1. 文件浏览器与沙箱文件系统集成增强（优先级 P1）

**当前问题**：`FileExplorer.vue` 功能基础，与沙箱文件系统的交互有限。

#### 改进项

- **实时文件树**：监听沙箱内文件变更，自动刷新目录树
- **文件预览**：点击文件直接在右侧面板预览内容（代码高亮）
- **拖拽上传**：支持从本地拖拽文件到文件浏览器上传至容器
- **右键菜单**：新建文件/文件夹、重命名、删除、下载
- **文件搜索**：在沙箱内按文件名或内容搜索
- **多文件选择**：批量下载/删除操作

---

### 2. 代码 Diff 查看器（优先级 P1）

**当前问题**：Agent 使用 `str_replace_editor` 工具修改文件时，用户只能看到原始的工具调用参数，无法直观理解变更内容。

#### 改进项

- **Diff 渲染组件**：在 `MessageBubble.vue` 或 `TimelineEventItem.vue` 中，当检测到 `str_replace_editor` 工具调用时，渲染 side-by-side 或 unified diff 视图
- **语法高亮**：diff 内容使用对应语言的语法高亮
- **折叠/展开**：大文件变更支持折叠未修改部分
- **一键复原**：提供"撤销此修改"按钮（调用反向 str_replace）
- 技术选型：`monaco-editor` 的 diff 模式或轻量级 `diff2html`

---

### 3. 计划模式可视化改进（优先级 P2）

**当前问题**：`PlanPanel.vue` 和 `TodoBoard.vue` 以列表形式展示计划步骤，缺乏直观的进度和依赖关系展示。

#### 改进项

- **DAG 视图**：将计划步骤渲染为有向无环图，展示任务依赖关系
- **进度条**：每个步骤显示执行进度（pending/running/done/failed）
- **时间线视图**：类 Gantt 图展示各步骤的时间分布
- **步骤详情浮窗**：悬停查看步骤的输入/输出/耗时
- 技术选型：`@vue-flow/core`（DAG）或 `gantt-schedule-timeline-calendar`

---

### 4. 设置面板改进（优先级 P2）

**当前问题**：`SettingsPanel.vue` 功能完整但缺乏便捷的配置管理。

#### 改进项

- **连接 Profiles**：保存多组 SSH + Docker 配置，快速切换（如"开发服务器"/"测试服务器"）
- **LLM Presets**：预设常用模型配置（OpenAI GPT-4o, Claude 3.5, 本地 Ollama 等），一键选择
- **配置导入/导出**：将完整配置导出为 JSON，支持在不同设备间迁移
- **连接测试**：每项配置（SSH/Docker/LLM/MCP）旁增加"测试连接"按钮
- **配置校验**：实时校验输入（如 SSH 端口范围、API URL 格式）

---

### 5. 主题定制（优先级 P3）

**当前问题**：硬编码暗色主题，无法切换。

#### 改进项

- **内置主题**：提供暗色/亮色/自动（跟随系统）三种预设
- **CSS 变量方案**：将所有颜色抽取为 CSS 自定义属性
  ```css
  :root {
    --bg-primary: #1a1a2e;
    --bg-secondary: #16213e;
    --text-primary: #e0e0e0;
    --accent: #0f3460;
  }
  ```
- **自定义主题**：用户可调整主要颜色，保存为配置
- **代码高亮主题**：与编辑器/diff 查看器的语法高亮主题联动

---

### 6. 快捷键支持（优先级 P2）

#### 核心快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Enter` | 发送消息 |
| `Ctrl+L` | 清空当前对话 |
| `Ctrl+N` | 新建会话 |
| `Ctrl+Tab` | 切换会话 |
| `Ctrl+,` | 打开设置 |
| `Ctrl+E` | 切换文件浏览器 |
| `Ctrl+T` | 切换终端面板 |
| `Ctrl+.` | 中断 Agent |
| `Escape` | 关闭弹窗/面板 |

- 使用 `@vueuse/core` 的 `useMagicKeys` 或自定义 composable
- 在设置中展示快捷键列表，后续支持自定义映射

---

### 7. 分栏布局改进（优先级 P3）

**当前问题**：`MainLayout.vue` 布局固定，无法灵活调整各面板大小。

#### 改进项

- **可拖拽分栏**：侧边栏、聊天面板、文件浏览器之间的分割线可拖动调整
- **面板折叠**：双击分割线折叠/展开面板
- **布局预设**：提供"对话优先"/"文件优先"/"均衡"等布局预设
- **面板浮动**：支持将文件浏览器或终端面板拖出为独立浮动窗口
- **布局记忆**：记住用户上次的面板大小和状态
- 技术选型：`splitpanes` 或自定义 resize 指令

---

## 实施优先级总结

| 改进 | 优先级 | 预估工作量 | 用户影响 |
|------|--------|-----------|---------|
| 文件浏览器增强 | P1 | 中 | 高 |
| 代码 Diff 查看器 | P1 | 中 | 高 |
| 快捷键支持 | P2 | 小 | 中 |
| 计划模式可视化 | P2 | 大 | 中 |
| 设置面板改进 | P2 | 中 | 中 |
| 主题定制 | P3 | 中 | 中 |
| 分栏布局改进 | P3 | 大 | 中 |
